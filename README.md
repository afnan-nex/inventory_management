# Inventory Management System - Phase 1

Production-quality Inventory Management System built with Node.js, Express, and SQLite.

## Installation

```bash
npm install
```

## Database Initialization

```bash
npm run seed
```

## Run

```bash
npm start
```

## Default Account

- Email: admin@example.com
- Password: admin123
- Role: Owner

## Features

- JWT Authentication
- Role-based access control (Owner, Manager, Employee)
- Responsive Bootstrap UI
- Secure session management
- Input validation
- SQL injection prevention
- Rate limiting
- Error handling

## Tech Stack

- Node.js + Express.js
- SQLite (better-sqlite3)
- JWT + bcrypt
- Bootstrap 5
- Helmet, CORS, express-validator
